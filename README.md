**<p align="center">Sistema de Monitoramento de Carros com ESP32</p>**
___________________________________________________________________________
Este projeto implementa um sistema simples de detecção e contagem de carros utilizando um sensor LDR conectado a um ESP32. Sempre que um carro é detectado, o sistema:

*Incrementa um contador*
*Exibe uma mensagem no terminal*
*Pisca um LED como indicação visual*
___________________________________________________________________________
**<p align="center">Funcionamento Geral📌 </p>**

*O sistema utiliza um sensor de luminosidade (LDR) para detectar a passagem de veículos. Quando há alteração na luz (indicando um carro), o sistema registra o evento.*
___________________________________________________________________________
**<p align="center">Hardware Utilizado⚙️ </p>**
*-ESP32*<br>
*-Sensor LDR (fotoresistor)*<br>
*-LED*<br>
*-Resistores (para o LED)*
___________________________________________________________________________
*<p align="center">*Configuração de Pinos*🔌</p>*

***Componente	GPIO:***<br>
*-LDR	25*<br>
*-LED	2*<br>
